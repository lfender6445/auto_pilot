# AutoPilot

Convert your stackoverflow.com profile to a github blog

## Installation

Add this line to your application's Gemfile:

```ruby
gem 'auto_pilot'
```

And then execute:

    $ bundle

Or install it yourself as:

    $ gem install auto_pilot
